MIUI 欧洲版指北

Posted on 2022-06-25

写在前面

国内版：自带应用太多，后台总是在跑一些乱七八糟的东西，电量消耗快，某些角落藏有广告，搜狗输入法无法卸载，默认应用总是被修改；欧洲版无广告、小米应用商店，自带 GMS、Play 商店、Gboard 输入法。更多优缺点欢迎在评论区补充。

工具准备

Windows (7+)电脑 * 1，受支持的小米手机 * 1，数据线 * 1，U 盘 * 1(OTG，可选)，以及可以自由冲浪的网络环境；

以小米 MIX 2S 为例，电脑上准备以下文件：

miflash_unlock-x.x.xxx.xx.zip–>(Bootloader 解锁工具)

UniversalAdbDriverSetup.msi–>(ADB 驱动)

platform-tools_rxx.x.x-windows.zip–>(ADB 调试工具)

twrp-3.5.2_9-0-polaris.img–>(TWRP Recovery)

xiaomi.eu_multi_XXXXX_V12.x.x.x.XXXXXXX_v12-xx.zip–>(MIUI 欧洲版 ROM 卡刷包)

解锁 Bootloader

对于未解锁 Bootloader 的设备，进入设置->我的设备->全部参数->点击[MIUI 版本]若干次直至提示“您现在处于开发者模式”；

进入“设置 -> 开发者选项 -> 设备解锁状态”中绑定账号和设备；获得解锁权限需要等待一段时间（一般 1~2 天）；

获得解锁权限后，手动进入 Bootloader 模式（关机后，同时按住开机键和音量下键）；

下载 MIUI 官网的解锁工具miflash_unlock-x.x.xxx.xx.zip->解压后打开miflash_unlock.exe->登录小米账号，通过 USB 连接手机，点击 “解锁”按钮；

解锁完成后备份重要数据，登出小米账号，移除所有 Google 账号。

配置环境变量

解压platform-tools_rxx.x.x-windows.zip得到 platform-tools 文件夹，打开设置> 系统> 高级系统设置 >环境变量，在系统变量中选择 Path，点击下面的编辑 > 新建 > 粘贴 D:\platform-tools(按照实际路径，不要照搬) > 确定。

安装 ADB 驱动

双击UniversalAdbDriverSetup.msi安装驱动，一直下一步，完成后重启电脑。

进入 Fastboot 模式

将twrp-3.5.2_9-0-polaris.img移动到 platform-tools 文件夹中；

确保手机 USB 调试已打开，进入设置 > 更多设置 > 开发者选项，打开 USB 调试，将手机连接到电脑的==USB2.0==接口（USB3.0 可能会报错）；

打开 Windows Powershell 或者 Command Prompt，输入 cd D:\platform-tools 进入文件夹，输入adb devices查看设备连接情况，输入adb reboot bootloader进入 Fastboot 模式，输入fastboot devices再次确认设备连接是正常的。

启动 TWRP Recovery

然后输入fastboot boot .\twrp-3.5.2_9-0-polaris.img让手机进入 TWRP Recovery，看到以下画面就说明成功进入 twrp 了，此时可将数据线移除，设置语言，不自动锁屏。

清除数据

点击 Wipe->Advanced Wipe->选中所有->滑动滑块确认清除；返回上一级，点击格式化 Data 分区，返回首页。

开始搞搞阵

插入 U 盘，点击安装

笔者这里使用 U 盘安装，没有 U 盘的朋友可以将 ROM 包拷到手机(/sdcard)中，然后选择从[内置存储]安装；

选择内置存储/USB-OTG，开始刷入，等待进度条走完，点击重启系统，在 MIUI 的 logo 界面耐心等待一段时间；一顿客制化后：

后台回复miui_eu获取相关链接，回复miui_eu_mix2s获取 MIX 2S 工具。

Enjoy!
