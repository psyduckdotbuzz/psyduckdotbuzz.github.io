MIUI 欧洲版指北

发表于 2022-06-25 更新于 2022-07-10 分类于 Android

MIUI 国内版有很多让人诟病的地方，比如自带小米全家桶和很多第三方 App、过多的广告、没有预装 Play Store 等，这篇文章手把手地教你刷入相对原生的 MIUI 欧洲版，改善使用体验。

准备

Windows（10+）电脑 * 1，受支持的设备 * 1，数据线 * 1，U盘 * 1（可选），以及可以自由冲浪的网络环境。电脑上准备对应设备的刷机工具：

  • miflash_unlock-xxx.zip —>（ Bootloader 解锁工具）

  • UniversalAdbDriverSetup.msi —>（ ADB 驱动）

  • usb_driver_xxx.zip —> （ USB 驱动）

  • platform-tools_xxx.zip —>（ ADB 调试工具）

  • twrp-xxx.img —>（第三方 Recovery ）

  • xiaomi.eu_multi_xxx.zip —>（ 欧洲版 ROM ）

解锁 Bootloader

对于未解锁 Bootloader 的设备，进入设置->我的设备->全部参数->点击 [ MIUI 版本] 若干次直至提示“您现在处于开发者模式”。

进入“设置 -> 开发者选项 -> 设备解锁状态”中绑定账号和设备，获得解锁权限需要等待一段时间。

获得解锁权限后，手动进入 Bootloader 模式（关机后，同时按住开机键和音量下键）。

下载MIUI官网的解锁工具 miflash_unlock-xxx.zip -> 解压后打开 miflash_unlock.exe -> 登录小米账号，通过 USB 数据线连接手机，点击“解锁”按钮。

解锁完成后备份重要数据，登出小米账号，移除所有Google账号。

配置环境变量

解压 platform-tools_xxx.zip 得到 platform-tools 文件夹，打开设置> 系统> 高级系统设置 >环境变量，在系统变量中选择Path，点击下面的编辑 > 新建 > 粘贴 D:\platform-tools(按照实际路径，不要照搬) > 确定。

安装 ADB 驱动

双击 UniversalAdbDriverSetup.msi 安装驱动，一直下一步，完成后重启电脑。

进入 Fastboot 模式

将 twrp-xxx.img 移动到 platform-tools 文件夹中。

确保手机 USB 调试已打开，进入设置 > 更多设置 > 开发者选项，打开 USB 调试，将手机连接到电脑的 USB 2.0 接口（ USB 3.0可能会报错），打开 Windows Powershell 或者 Command Prompt，进入文件夹：

查看设备连接情况：

如果没有反应，进入设备管理器，有名为的黄标，先安装 USB 驱动并重启电脑。

进入Fastboot模式：

查看设备连接状态：

启动 TWRP

然后输入 fastboot boot .\twrp-xxx.img 让手机进入TWRP Recovery，看到以下画面就说明成功进入 twrp 了，此时可将数据线移除，设置语言，不自动锁屏。

清除数据

点击 Wipe -> Advanced Wipe -> 选中所有->滑动滑块确认清除；返回上一级，点击格式化 Data 分区，返回首页。

刷入

插入 U 盘，点击安装

笔者这里使用U盘安装，没有 U 盘的朋友可以将 ROM 包拷到手机（/sdcard）中，然后选择从[内置存储]安装。

选择内置存储/USB-OTG，开始刷入，等待进度条走完，点击重启系统，在 MIUI 的 logo 界面耐心等待一段时间。
